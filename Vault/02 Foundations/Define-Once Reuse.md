# Define-Once Reuse

> A concept defined once in the model means the same thing in every screen, query, rule, and agent.

An object, property, link, function, or action is authored once and reused everywhere. One definition → every app; one logic → every surface; one permission model → every access path. This is the core payoff of a [[Semantic Model Over Data|semantic layer]].
