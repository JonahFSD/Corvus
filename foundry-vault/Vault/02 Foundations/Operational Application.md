# Operational Application

> An app built to drive a decision and capture it via writeback — as opposed to a read-only dashboard.

Where a dashboard delivers read-only insight, an operational application lets users take an [[Actions#Action Type|action]] and commit the decision back to the [[Ontology & Metadata#Ontology|Ontology]]. This is the dividing line between Foundry and ordinary BI.
