# RID

> Resource Identifier: Foundry's universal addressing scheme, ri.<service>.<instance>.<type>.<locator>.

Every resource — dataset, transaction, object type, action type, source, build — has a stable RID. The naming substrate that makes [[End-to-End Lineage|lineage]] and references possible.
