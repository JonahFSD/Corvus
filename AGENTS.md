# Corvus

Corvus is a greenfield product repository operated with the Codex-native agentic engineering workflow.

- Package manager: npm.
- Run `npm run check` before declaring implementation work complete.
- Build behavior through tested vertical slices; review every diff against both its ticket/spec and repository standards.
