# Triage labels

| Canonical role    | GitHub label      | Meaning                                         |
| ----------------- | ----------------- | ----------------------------------------------- |
| `needs-triage`    | `needs-triage`    | A maintainer must classify the request.         |
| `needs-info`      | `needs-info`      | Work is waiting for missing information.        |
| `ready-for-agent` | `ready-for-agent` | Fully specified AFK work.                       |
| `ready-for-human` | `ready-for-human` | Requires live human judgment or implementation. |
| `wontfix`         | `wontfix`         | Deliberately will not be actioned.              |

Workflow labels supplement, but never replace, these roles. AFK automation keys only on `ready-for-agent`.
